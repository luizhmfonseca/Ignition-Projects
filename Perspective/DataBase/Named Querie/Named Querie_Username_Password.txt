## No SSMS / Named Query (SQL Server)
##Certifique-se de que sua Named Query Login_Users_Open_Tickets esteja configurada assim no editor do Ignition, 
pois o SQL Server é rigoroso com tipos de dados:
## Query Type: Scalar Query (se quiser apenas um valor) ou Query (para retornar a linha toda).
## Parâmetros: username (String) e password (String).

Código SQL:


SELECT username 
FROM scada_user_source_users 
WHERE username = :username 
  AND password = :password
